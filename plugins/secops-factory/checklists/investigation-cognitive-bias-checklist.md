# Investigation Cognitive Bias Detection Checklist

**Weight:** 5%
**Purpose:** Verify objective analysis free from cognitive biases.

## Check Items

- [ ] No confirmation bias (considered disconfirming evidence)
- [ ] No anchoring bias (not locked to initial alert severity without assessment)
- [ ] No availability bias (not over-weighting recent incidents without statistics)
- [ ] No recency bias (considered 30/60/90 day historical patterns)
- [ ] No automation bias (independently verified platform disposition/severity)
- [ ] Alternative hypotheses considered (at least 2 alternatives evaluated)
