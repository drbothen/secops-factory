# Investigation Technical Accuracy Checklist

**Weight:** 20%
**Purpose:** Verify factual correctness and technical validity of all claims.

## Check Items

- [ ] IP addresses and network identifiers correct (valid ranges, no typos)
- [ ] Protocol and port information accurate (correct names, valid ranges 1-65535)
- [ ] Alert signature/rule correctly identified (matches platform documentation)
- [ ] Technical terminology used correctly
- [ ] Log excerpts interpreted correctly (timestamps, fields, format)
- [ ] Attack vectors described accurately (technically feasible, logical)
- [ ] No contradictions between evidence and conclusions
