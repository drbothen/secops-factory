# Investigation Documentation Quality Checklist

**Weight:** 5%
**Purpose:** Verify clear, professional, structured documentation.

## Check Items

- [ ] Logical structure and flow (Alert -> Context -> Investigation -> Evidence -> Disposition -> Actions)
- [ ] Professional tone and language
- [ ] Minimal typos/grammatical errors (fewer than 5)
- [ ] Key findings highlighted or summarized
- [ ] Evidence references clear and verifiable (timestamps, source systems)
- [ ] Timestamps in consistent format (preferably UTC)
